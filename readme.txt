/*******************
* Youtube Repeater *
*******************/

Youtubeのサイト上でリピート再生をするブックマークレットです。
youtube_repeater_bookmarklet.jsの中身を他のブックマークレット作成と
同じようにブックマークに登録してください。

後はYoutubeを開いた状態でクリックして頂くと、動画が自動的にリピートされます。

参考
Youtube JavaSript Player APIリファレンス
https://developers.google.com/youtube/js_api_reference?hl=ja
